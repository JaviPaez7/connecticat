# Connecticat

Plataforma web para amantes de los gatos — Catstagram, Gatinder, Curiosidades y Purr-Foro.

**Producción:** [connecticat.javistudio.dev](https://connecticat.javistudio.dev)

## Stack

- **Astro 7** — SSG (curiosidades) + islas React (zonas interactivas)
- **React 19 + TypeScript** — UI interactiva
- **Tailwind CSS v4** — design system “Sunrise Whiskers”
- **PostgreSQL + Prisma** — datos relacionales (Paso 2)
- **Lucide React + Framer Motion** — iconos y motion

## Desarrollo

```bash
npm install
npm run dev
```

```bash
npm run build
npm run preview
```
